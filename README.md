# DC — Comic Book Reader

A comic book reader app with smooth zoom support for the most popular comic formats.

## Supported Formats

| Format | Description |
|---|---|
| CBZ | ZIP-based comic archive (most common) |
| CBR | RAR-based comic archive |
| CB7 | 7-Zip-based comic archive |
| CBT | TAR-based comic archive |
| PDF | Standard PDF documents |
| EPUB | Reflowable ebook format (image-based comics) |

## Key Features (planned)

- Open and navigate CBZ, CBR, CB7, CBT, PDF, EPUB
- Smooth pinch-to-zoom and scroll-to-zoom with high-quality rendering
- Double-tap to zoom to panel / fit-to-width / fit-to-page
- Page turn: swipe or keyboard navigation
- Reading modes: single page, double page spread, vertical scroll
- Bookmarks and reading progress persistence
- Library view with cover thumbnails
